# Install Dev Tools

Цей репозиторій містить Bash-скрипт для автоматичного встановлення основних інструментів DevOps на Ubuntu/Debian.

Скрипт встановлює:
- Docker
- Docker Compose
- Python (3.9+)
- Django

Та перевіряє, чи вони вже встановлені.

---

## Як запустити

```bash
chmod +x install_dev_tools.sh
./install_dev_tools.sh
